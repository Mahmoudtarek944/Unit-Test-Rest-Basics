# JavaScript Unit Testing 
This repository documents my journey learning JavaScript testing using Jest.
## What I focused on
- Understanding how JavaScript works under the hood
- Writing reliable and testable functions
- Using mocking to simulate real scenarios
- Measuring test coverage
- Handling async operations (Promises, fetch)
  
## Structure
- Implementation (function)
- Test file in Folder tests
- Real examples & edge cases
- Coverage Folder 
## Goal
To move from "code that works" to "code that is reliable and testable"
